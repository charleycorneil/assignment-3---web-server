About This Project

This project is a simple Node.js web server built for an assignment. It demonstrates how to:

  • HTML, CSS, and JavaScript files
  • Use the `http` and `fs` modules to handle file requests
  • Implement navigation between pages
  • Serve media files such as images
  • Handle 404 errors gracefully

The web server runs on port 3000 and serves files dynamically based on the requested URL.
